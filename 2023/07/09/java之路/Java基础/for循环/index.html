for循环

这是最基本的for循环形式，for循环的格式为for(初始化;布尔表达式;更新){}，for循环语句是支持迭代的一种通用结构，是最有效、最灵活的循环结构。for(;;){}是for循环的死循环。

for循环例子

  1. 计算0-100之间奇数和偶数的和
  1. 输出1-1000之间能被5整除的数，并且每行输出三个
  1. 打印九九乘法表
