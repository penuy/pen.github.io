MarkDown语法

MarkDown语法

标题

三级标题

四级标题

几级标题即几个#号加空格后加标题名称（支持六级标题）

字体样式

**hello,world!*（字体加粗即在要加粗字体两侧加两个）

hello,world! （斜体即在要变为斜体的部分两侧加一个*）

hello,world! （斜体加粗，即两侧加三个*）

hello,world!（字体中间划线即两侧加两个波浪号~~）

超链接

百度

超链接即中括号内写标题后跟括号写链接

表格

  1. 可直接使用Typora右键导入表格

  2. 输入表头（如：名字，年龄，性别）

    名字|年龄|性别

    –|–|–|

    后接内容，格式与表头保持一致（李明|28|男），后进入源代码模式，使其变为一行后退出源代码模式即可。

姓名 年龄 性别
李明 28 男

列表

有序即数字加点加空格

  1. 第一
  2. 第二行

无序即减号直接加空格

  • apple
  • banana

引用

MarkDown

引用就直接大于号加空格。

分割线

—或三个***加回车

（三个—）

（三个***）

图片

本地图即感叹号加中括号（中括号内写图片名字）加括号（括号内写图片存储地址地址）

网络图片即把地址写为图片网络链接。

代码

三个```后加所使用编码语言即可编写代码块。

hello

也可在两侧加`写代码块。
